Vždy inicializuj proměnnou!
```c
int i = 42;
```
- Poté dochází k divným věcem, když printujeme

##### Datové typy
int, float, double, char

##### printf
- `%d` - integer
- `%f` - floating point
- `%x` - hex
```c
int variable1 = 3;
int variable2 = 5;

printf("%d,   %d", variable1, variable2);
```
